# DrawSports Web

Web oficial de [DrawSports](https://drawsports.app) - Videoanálisis táctico para iPad.

## Estructura

- `Web/index.html` - Versión español
- `Web/en/` - Versión inglés
- `Web/password-updated/` - Página de confirmación de recuperación de contraseña (PRO)

## Despliegue

Desplegado en Vercel. Configurar **Root Directory**: `Web`.
